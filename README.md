# Bento-Box-Neo

*The fourth robot by Bento Robotics, a distillation of what worked well on the [Bento-Box](https://github.com/Bento-Robotics/Bento-Box).*  
This repo, it's [wiki](https://github.com/Bento-Robotics/Bento-Box-Neo/wiki) and the [CAD files on onshape](https://cad.onshape.com/documents/c218e97fcbfc8c36f1dda8f3/w/3e578e5385c8ade5279b02bd/) contain all information needed to rebuild Bento Box Neo from scratch.

## [system-files](./system-files/README.md)
All system software changes made on Bento Box Neo's computer.  
*networking, software installation*

## [container](./container/README.md)
The container system for robotics software.  
*ROS, fixing hangups*
